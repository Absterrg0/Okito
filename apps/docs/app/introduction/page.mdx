# Welcome to Okito

Okito is a set of library functions and modern UI components developed with the sole aim to reduce the developer complexity of Solana.


## What is Okito?

Okito currently has 2 powerful packages that help you accelerate the development with customizable functions:

- **@okito/sdk**: Core blockchain functions exposed for developing blockchain specfic transactions
- **@okito/ui**: Beautiful, customizable UI components to build stunning interfaces with just one component.

##  Why Choose Okito?

- **Developer Experience**: Exposes function for seasoned web3 developers reducing redundant code by a ton.
- **Performance**: Optimized for speed and efficiency while making sure the transaction confirms with the retry mechanism.
- **Flexibility**: Developed for NextJs and React projects for now.
