## `estimateAirdropFees`

This function is defined to calculate the fees required while airdropping a token(SPL) on the blockchain.

## Snippet

``` jsx filename="page.tsx"
import { useConnection } from "@solana/wallet-adapter-react";
import { estimateAirdropFee, type AirdropRecipient } from "@okito/sdk";

const destinationAddress:AirdropRecipient[] = [
  {
  address:"",
  amount:10,
  },
  {
  address:"",
  amount:20,
  },
  

]

const {connection} = useConnection();
const airdropFees = await estimateAirdropFee(connection,destinationAddress,10,0);
```
<Callout type='warning'>
<b>Note:</b> The priority fees is optional which can be paid to improve the chances of successfully passing the transaction.
</Callout>
