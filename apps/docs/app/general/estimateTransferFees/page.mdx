## `estimateTokenTransferFee`

This function is defined to calculate the value of the estimated fee while transferring a token(SPL) on the blockchain.

## Snippet

``` jsx filename="page.tsx"
import { useConnection } from "@solana/wallet-adapter-react";
import { estimateTokenTransferFee, getMintAddress } from "@okito/sdk";

const {connection} = useConnection();
const usdcMintAddress = getMintAddress("USDC",'devnet').toString();
const tokenTransferFees = await estimateTokenTransferFee(connection,usdcMintAddress,recipientAddress,0)
```
<Callout type='warning'>
<b>Note:</b> The priority fees is optional which can be paid to improve the chances of successfully passing the transaction.
</Callout>
