# `estimateTokenCreationFee`

Estimates rent and fees for creating a new token.

<Callout type="info">
The estimate includes rent for the mint + metadata account and an RPC-provided fee estimate for a representative transaction message (create mint, init metadata pointer, init mint, init metadata, create ATA, mint initial supply).
</Callout>

## Usage

```jsx filename="page.tsx"
import { useConnection } from "@solana/wallet-adapter-react";
import { estimateTokenCreationFee, type TokenData } from "@okito/sdk";

const { connection } = useConnection();

const tokenData: TokenData = {
  name: "My Test Token",
  symbol: "TEST",
  imageUrl: "https://example.com/token-image.png",
  decimals: 9,
};

const feeInfo = await estimateTokenCreationFee(connection, tokenData);
```
