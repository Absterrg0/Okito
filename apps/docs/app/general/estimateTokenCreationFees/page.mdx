## `estimateTokenCreationFees`

This function is defined to calculate the value of the estimated fee while creating a token(SPL) on the blockchain.

## Snippet

``` jsx filename="page.tsx"
import { useConnection } from "@solana/wallet-adapter-react";
import { estimateTokenCreationFee, type TokenLaunchData } from "@okito/sdk";

const tokenData:TokenLaunchData ={
  name:"TEST",
  symbol:"TST",
  imageUrl:"https://example.com/image.png",
  initialSupply:1000000000,
  decimals:6,
  freezeAuthority:true,
} 

const {connection} = useConnection();
const tokenCreationFees = await estimateTokenCreationFee(connection,tokenData,0)
```
<Callout type='warning'>
<b>Note:</b> The priority fees is optional which can be paid to improve the chances of successfully passing the transaction.
</Callout>
