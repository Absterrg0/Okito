## `getMintAddress`

Resolves a token symbol to its mint address for a given network.

## Usage

``` jsx filename="page.tsx"
const mintAddress = getMintAddress("USDC",'mainnet-beta')
```
<Callout type='warning'>
<b>Note:</b> This function currently only supports **USDC** and **USDT** tokens. More tokens will be added soon.
</Callout>

    { name: "network", type: '"mainnet-beta" | "devnet"', description: "Network to use.", default: '"mainnet-beta"' },
  ]}
/>
