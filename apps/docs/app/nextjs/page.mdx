# Next.Js


Install and configure @Okito packages for Next.Js project



<Steps>
### Step 1

Create a new Next.Js project by running the following command:

```sh filename="terminal"
npx create-next-app@latest 
```
<Callout type="info"> Okito SDK package is developed for both TS and JS projects</Callout>

### Step 2

Install the necessary **@oktio/sdk** package:

```sh filename="terminal"
npm install @okito/sdk
```
