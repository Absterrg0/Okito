# `burnToken`

This function allows you to burn a token on the Solana blockchain.
## Snippet

``` jsx filename="page.tsx"
import { useConnection, useWallet } from "@solana/wallet-adapter-react";
import { burnToken } from "@okito/sdk";

const {connection} = useConnection();
const wallet = useWallet();

const mint = "";            // The mint address of the token to be burned
const amount = 1;           // The amount of the token to be burned


const burnResult = await burnToken(connection,wallet,mint,amount)

```
