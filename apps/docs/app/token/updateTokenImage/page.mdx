# `updateTokenImage`

This function allows you to update the image (URI) of an existing token on Solana using the Okito SDK.
## Usage

``` jsx filename="page.tsx"
import { useConnection, useWallet } from "@solana/wallet-adapter-react";
import { updateTokenImage } from "@okito/sdk";

const {connection} = useConnection();
const wallet = useWallet();
const mintAddress = ""; //mint address for the token 
const newImageUrl = ""; //the new image URL

const updateResult = await updateTokenImage(connection,wallet,mintAddress,newImageUrl)

```
