## `getTokenBalanceBySymbol`

This function allows you to fetch the token balance of a token for any address provided

## Snippet

``` jsx filename="page.tsx"
import { useConnection } from "@solana/wallet-adapter-react";
import { getTokenBalanceBySymbol } from "@okito/sdk";

const {connection} = useConnection();

const mintAddress = ""; //mint address for the token 
const for = "";  // the account for which balance is requested 
const tokenBalance = await getTokenBalanceBySymbol(connection,for,mintAddress)
```

<Callout type="warning">
  This function will only works  for fetching token balances of tokens that can be fetched using `getMintAddress`.
</Callout>
