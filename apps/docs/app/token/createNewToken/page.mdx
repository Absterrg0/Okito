## `createNewToken`

This function is defined to launch a new token on the blockchain. The launched token follows the SPL token standard.

## Snippet

``` jsx filename="page.tsx"
import { useConnection } from "@solana/wallet-adapter-react";
import { createNewToken } from "@okito/sdk";

const {connection} = useConnection();
const wallet = useWallet();

const tokenTransferFees = await createNewToken(connection,wallet,tokenData)
```
<Callout type='warning'>
<b>Note:</b> The priority fees is optional which can be paid to improve the chances of successfully passing the transaction.
</Callout>
